## 🎨 项目结构

```
├── majiang-server         # 麻将服务端
├── core                   # 核心模块
├── migrations             # sql操作脚本
├── models                 # 表建模
├── routers                # api路由
├── services               # 游戏相关服务
├── utils                  # 工具类
├── .gitignore             # git忽略配置
├── package.json           # 依赖表
└── tsconfig.json          # ts配置文件
```
